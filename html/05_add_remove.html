Page 5: Adding and Removing Items from an Array

Sometimes you need to add additional items to an array or remove some of the existing item.

To add an item to an array, you can use unshift or push. Unshift adds an item to the beginning of the array and shifts all the other items to a higher index number. Push adds an item to the end of the array. The syntax is:
array_name.unshift();
array_name.push();

If you need to remove an item from an array, you can use shift or pop. Shift removes the first item in the array, and pop removes the last item in the array. The syntax is:
array_name.shift();
array_name.pop();

You can also add and remove items from an array using splice() and slice(). These methods are actually more flexible than shift, pop, unshift, and push because they allow you to add and remove the items from any location in the array.

The syntax for splice(), which adds items to an array is:

array_name.splice(2, 0, "item1", "item2");

Note that the first parameter, 2, is the position in the array where the items should be added, and the second parameter, 0, is how many items should be removed. The remaining parameters are the things that will be added to the array. You can adjust these parameters to your needs.

The syntax for slice(), which removes items from an array is:

array_name.slice(2);

Note that parameter, 2, is the position in the array before which the items should be removed. A new array will be returned that does not include any of the items with an index less than 2.
