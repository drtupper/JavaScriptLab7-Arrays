Page 1: Create an Array

Arrays are ideal for storing lists of items because they allow you to access the items using a single variable name and an index number.

To declare an array variable, use the keyword const. The array variable itself cannot be reassigned to a new array, but the items in the array can be changed.

The syntax to create an array is:
const array_name = [item1, item2, item3, . . . .]
