Page 3: Changing Array Items

While array variables cannot be reassigned, the individual array items can be changed. The syntax for this is:

array_name[index_number] = new_value;

For example, cars[0] = "Volvo"); will change the first item in the array to "Volvo".

1. Go to the 03_change.html file and link the 03_change.js file. You will know that you've done this correctly if you see the statement "03_change.js linked" when you refresh this page

2. Go to 03_change.js file and follow instructions to change the values of some of the items in an array.
