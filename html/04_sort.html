Page 4: Sorting an Array

The sort() method can be used to arrange the items in an array in alphabetical order.

The syntax for sorting is:
const fruits = ['Banana', 'Orange', 'Apple', 'Mango']
fruits.sort();

This will change the array fruits to ['Apple', 'Banana', 'Mango', 'Orange']
