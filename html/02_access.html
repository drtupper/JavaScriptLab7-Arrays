Page 2: Accessing Array Items

How do you access items stored in the array once it is created? The items can be accessed by typing the name of the array, followed immediately by their index number in square brackets. Items are numbered in order, starting at 0. The first item in the array is array_name[0]

1. Go to the 02_access.html file and link the 02_access.js file. You will know that you've done this correctly if you see the statement "02_access.js linked" when you refresh this page

2. Go to 02_access.js file and follow instructions to practice accessing items from arrays.
